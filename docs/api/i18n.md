# i18n API Reference

Localization support for help text and error messages.

## Functions

### `argparse_set_locale`

```c
void argparse_set_locale(const char *locale);
```

Set the active locale for translated messages. Default is "en" (English). Not thread-safe (global state).

### `argparse_get_locale`

```c
const char *argparse_get_locale(void);
```

Get the current locale string. Safe to read concurrently.

### `argparse_i18n_get`

```c
const char *argparse_i18n_get(const char *key);
```

Get a translated string for a known message key. Falls back to English, then to the key itself. Safe to read concurrently.

### `argparse_i18n_set`

```c
int argparse_i18n_set(const char *locale, const char *key,
                       const char *translation);
```

Register a translation for a key in a specific locale. Returns 0 on success, -1 on failure. Not thread-safe.
